Uses

This is list of my personal devices and preferred applications and services that I am currently using for development purposes.

Workstation

  • 13" MacBook Air 3.2ghz M1, 16GB RAM, 256GB Storage (2020)

    Prior to this I was using a custom PC that I specifically built for gaming. Over the years my passion for gaming had declined, so I decided to trade my PC in a favor for a portable machine.

  • 14" HP ProBook 440 G7 4.9ghz i7-10510U, 20GB RAM, 512GB Storage (2020)

    This machine is provided by my current employer which I requested for developer specifications. I run Windows and Linux on this machine to develop work related projects.

  • 34" Acer Nitro XV340CKP 144hz, 3440x1440 Monitor

    Switching from 2 monitors setup to an ultrawide monitor has greatly improved my productivity. I personally prefered non curved over curved displays generally because of design aesthetic.

  • Keychron K8 Gateron Brown Switch Keyboard

    I favor TKL design over full sized keyboard because I never have the need to use numeric keypad in my entire life. I choose brown switch due to its tactile bump but silent travel.

  • Xiaomi 1300dpi Dual Mode Wireless Mouse

    I use my trackpad and keyboard shortcuts for travesing a lot more simply because its faster. On some rare occasions when I am using a mouse, this budget mouse fits the need.

Development Tools

  • Visual Studio Code

    I consider this as the GOAT fot its various plugin extensions, built-in git repository tracking and integrated terminal. Presonally I prefer the default dark theme.

  • TablePlus

    My go-to DBMS when managing database for its intuitive and sleek design. Its cross-platform compatibility ensures seamless usage across multiple operating systems.

  • Docker

    A must have tool for every developer. This platform eliminated my need for complex virtualization, simplifying the process of setting up my development environment.

  • Postman

    Another must have tool for every developer. This API platform allows me to efficiently build and test APIs during development, streamlining the creation process and enhancing productivity.

  • Github

    This version control platform is where I store all of my source codes, ensuring version history is tracked and safeguarded. I don't know what I would do without it.

  • Chrome, Safari, Edge, Fire and Opera Web Browsers

    I mainly use Chrome for browsing on all my devices. Other browsers are for debugging purposes as different browsers have their respective set of behaviors.

Hosting Services

  • Digital Ocean

    This is my chosen platform for deploying the majority of my core projects. Its scalability and containerization abilities makes deployment to be effortless.

  • Vercel

    This is my preferred hosting platform show project demonstration. Its seamless deployment process and most importantly free tier perfectly suits all my requirements.

  • Github Pages

    I host most of my static projects on Github Pages including this site. Deployment can be done by simply pushing my code into the repository.

Notable Applications

  • Figma

    Simple but exceptional design tool that I rely on for wireframing, prototyping, and even logo creation. Its user-friendly interface and intuitive features make it easy to bring my design ideas to life.

  • Notion

    Note-taking application that I rely on for organizing my thoughts, planning my next steps, and keeping track of important information. I installed application accross all my devices.

  • Spotify

    I code better algorithms while listening to music. I find that listening to synthwave, lofi, and trance genres creates the perfect ambiance for programming.
