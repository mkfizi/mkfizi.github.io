Uses

This is list of my personal devices and preferred applications that I am currently using for development purposes.

Workstation

  • 13" MacBook Air 3.2ghz M1, 16GB RAM, 256GB Storage (2020)

    Prior to this I was using a mid range custom rig PC that I specifically built for gaming. Over the years my passion for gaming had declined, so I decided to traded my old PC in a favor for a portable machine.

  • 14" HP ProBook 440 G7 4.9ghz i7-10510U, 20GB RAM, 512GB Storage (2020)

    This machine is provided by my current employer which I requested for developer specifications. I run Windows and Linux on this machine to develop work related projects.

  • 34" Acer Nitro XV340CKP 144hz, 3440x1440 Monitor

    Switching from 2 monitors setup to an ultrawide monitor greatly improved my productivity. I personally prefered non curved over curved displays generally because of design aesthetic.

  • Keychron K8 Gateron Brown Switch Keyboard

    I favor TKL design over full sized keyboard simply because I never have the need to use numeric keypad in my entire life. I choose brown switch due to it's tactile bump but silent travel.

  • Xiaomi 1300dpi Dual Mode Wireless Mouse

    I use my trackpad and keyboard shortcuts for travesing a lot more than using a mouse. For some rare occasions, this budget mouse fits the need.

Development

  • Visual Studio Code

    I have tried various text editors before I settled on this one. It's various plugin extensions, built-in git repository tracking and integrated terminal made me to consider this as the GOAT. I use the default dark theme.

  • TablePlus

    This is my go-to DBMS for it's intuitive and sleek design. It's cross platform compatibility means I can use across my multiple operating systems.

  • Docker

    A must have tool for every developer. I never have to use any virtualization just to setup development environment after I started using this.

  • Postman

    Another must have tool for every developer. This API platform allows me to test and build APIs during development.

  • Github

    This version control platform is where I store all of my codes. I don't know what I would do without it.

  • Chrome, Safari, Edge, Fire and Opera Web Browsers

    I mainly use Chrome for browsing on all my devices. Other browsers are for debugging purposes as different browsers have their respective set of behaviors.

Hosting

  • Digital Ocean

    I use this to host most of my main projects. It's scalibility and easy to deploy docker containers makes deployment effortless.

  • Vercel

    This is where I host my demo projects. I use this simple because it's free for basic usage.

  • Github Pages

    I host most of my static projects on Github Pages including this site. Deployment can be done by simply pushing my code into the repository.

Other Applications

  • Figma

    Simple design tool that I use for wireframing and prototyping. Sometimes I use this to create logo for my personal projects.

  • Notion

    I put every notes and planning my next steps in this great note taking application. I installed application this on all my devices.
